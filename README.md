# Automated Finger X-Ray Report Generation using Deep Learning

## Project Overview
This project focuses on developing an automated system for analyzing finger X-ray images using Deep Learning and Computer Vision techniques. The goal is to assist medical professionals by providing faster preliminary analysis and supporting the process of medical report generation.

## Methodology
The project follows a deep learning pipeline that includes:

1. Dataset preprocessing and organization
2. Image augmentation for better model generalization
3. Feature extraction using a DenseNet121 Convolutional Neural Network (CNN)
4. Transfer learning to improve model performance
5. Model evaluation using validation metrics
6. Text processing for automated medical report generation

## Model
The system uses a **DenseNet121-based CNN model** for feature extraction and classification of finger X-ray images. Transfer learning is applied using a pretrained model to improve accuracy and reduce training time.
